# SpectraOps Logo Usage Guidelines

## Logo Versions
1. **Primary Logo**: Full logo with icon and text
2. **Icon Only**: For social media profiles and favicons
3. **Text Only**: For minimal space requirements
4. **Monochrome**: For single-color applications

## Color Palette
- **Primary**: #6366f1 (Indigo)
- **Secondary**: #8b5cf6 (Purple)
- **Accent**: #ec4899 (Pink)
- **Text Dark**: #1f2937
- **Text Light**: #f9fafb

## Minimum Sizes
- **Digital**: 120px width minimum
- **Print**: 1 inch width minimum
- **Favicon**: 32x32px

## Clear Space
- Maintain clear space equal to the height of the "S" icon around the logo
- Never place logo over busy backgrounds without sufficient contrast

## Do's and Don'ts
✅ Use approved color variations
✅ Maintain proportions
✅ Ensure sufficient contrast
❌ Don't stretch or distort
❌ Don't change colors arbitrarily
❌ Don't add effects or shadows
